Customer Credit Control - Trial

Amount Credit Control, Days Credit Control, Tolerance and Warnings.

This application gives you an idea what our "Customer Credit Control" application offers. Customer Credit Control Trial application allows User / Business to have complete control over credit sales done to a customer. These configurations can be done individually or Universally. Once configuration is done, system will take care of all the processes. Everything is automated and User / Business Does not have to take care of anything manually. We also have given tolerance if user / business wants to let go credit limit check to a certain level.

Customer Configuration

When you go to customer form, you can see the customer's credit control configuration. This configuration applies to that particular customer only. Here, users can configure credit control amounts, credit control days, tolerance, and warning email limits. Since this is a trial app, users can only use the amount credit control feature. Other functionalities are not enabled in this application.

Trial Application Warning

This is a trial application warning message. This warning will pop up when the user tries to use the functionalities which are not enabled in the trial version of customer credit control application like tolerance, warning, days credit limit, etc.

Universal Configuration

This is universal customer credit control. Fields are similar as customer’s form credit control. The only difference is customer credit control configured here will apply to all the customers in the system. except those who have credit control configured individually. Here you can see a checkbox named “Force Universal Credit Control”. It allows the overwriting of universal credit control configurations over individual credit control configurations. If it is checked true, the customer's individual credit control will be ignored and universal credit control will apply on them like all the other customers
